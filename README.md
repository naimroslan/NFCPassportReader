# nfc-passport-reader

A React Native library for reading ePassport using NFC for Android

## Installation

```sh
npm install nfc-passport-reader
```

## Usage

```js
import { multiply } from 'nfc-passport-reader';

// ...

const result = await multiply(3, 7);
```
